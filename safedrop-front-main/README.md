# SafeDrop Frontend

SafeDrop is a Web3 security platform with email-code sign-up, 2FA onboarding, and vault/grind verification flows.

## Routes (Planned / In-Progress)
- `/sign-in`
- `/sign-up`
- `/verify/vault`
- `/verify/grind`
- `/partner`

## Local Development

```bash
cd safedrop-front-main
npm install
npm run dev
```

## Environment

```
NEXT_PUBLIC_API_SERVER_URL=http://localhost:3001
NEXT_PUBLIC_PROJECT_ID=your_walletconnect_id
```
